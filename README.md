# KayscapeService
Kayscape — фронтенд-разработка сервиса бронирования жилья. Семантическая верстка, CSS Grid, интеграция с Supabase/PostgreSQL. Учебный проект (Hexlet College).
Фронтенд-приложение для поиска и аренды жилья. Проект разрабатывается в рамках производственной практики в Hexlet College. Основной фокус — создание расширяемой архитектуры интерфейса и работа с внешними данными.

## Технологический стек
* **Layout:** HTML5 (семантика), CSS3 (Grid Layout, Flexbox, Custom Properties).
* **Logic:** JavaScript (ES6+), асинхронное взаимодействие с API.
* **Database & Backend:** Supabase (PostgreSQL) для хранения данных о недвижимости и управления состоянием избранного.

## Реализованные решения
На текущем этапе заложена база интерфейса:
* Разработана структура основных разделов через семантические блоки `section` и `article`.
* Спроектирована сетка карточек объектов с поддержкой динамических индикаторов («Выбор гостей», «Новое»).
* Подготовлена архитектура стилей для поддержки «мягкой» визуальной концепции (beige-white палитра).

## Roadmap
Проект находится в активной фазе разработки. Планируемые этапы:

1.  **Интеграция БД:** Перенос статических данных из HTML в таблицы PostgreSQL на стороне Supabase. X
2.  **Динамический рендеринг:** Реализация логики на JS для автоматической генерации карточек на основе ответов от API. X
3.  **Система фильтров:** Разработка функционала сортировки объектов по категориям (квартиры, отели, дома). X 
4.  **User State:** Добавление возможности авторизации и сохранение понравившихся объектов в wishlist. X 
5.  **Оптимизация:** Адаптация под мобильные устройства и проверка доступности (A11y). X
